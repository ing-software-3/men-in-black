# Requisiotos funcionales
## Adecuacion funcional
RF01. El sistema debe permitir gestionar el inventario de productos
RF02. El sistema debe permitir registrar nuevos productos
## Eficiencia de desempeño
RF03. La pagina debe mostrar en 2 segundos o menos las busquedas de productos realizadas
## Compatibilidad 
RF04. El sistema debe permitir exportar los logs del programa
## Usabilidad
RF05. El sistema debe permitir filtrar los datos dependiendo las caracteristicas de la prenda
## Fiabilidad
RF06. El sistema debe crear logs de las ventas realizadas y modificaciones
## Seguridad
RF07. El sistema debe validar usuarios para ingresar al modulo de ventas
## Mantenibilidad
RF08. El sistema debe permitir actualizar precios de prendas sin afectar registros o al funcionamiento del aplicativo
## Portabilidad
RF09. El sistema debe funcionar correctamente en diferentes versiones de windows y linux
